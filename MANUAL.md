## 1. 처음 시작할 때

### 프로그램 처음 설치하는 경우

1. `리와인드자동화.exe` 실행
2. 상단 **⚙** 버튼 클릭 → 아래 항목 입력 후 저장

   | 항목 | 입력 내용 |
   |------|----------|
   | 총매출 파일 비밀번호 | 총매출 엑셀 파일 암호 |
   | 지출 시트 이름 | 기본값 `데일리지출` (변경 안 했으면 그대로) |
   | 센터 월 목표금액 | 숫자만 입력 (예: `10000000`) |
   | 피티 월 목표금액 | 숫자만 입력 |

3. 메인 화면으로 돌아와서 **📂 버튼** 눌러 데일리 파일, 총매출 파일 각각 등록
   - 또는 탐색기에서 데일리 파일을 프로그램 창으로 드래그해도 됩니다

> 한 번 등록해두면 다음부터는 프로그램 켤 때 자동으로 불러옵니다.

### 매일 출근해서 프로그램 켤 때

별도로 할 것 없이 프로그램을 실행하면 됩니다.  
오늘 날짜 데일리 파일이 없으면 자동으로 만들어주고, 메인 화면에 오늘 매출이 바로 표시됩니다.

## 2. 결제가 들어왔을 때

**[💳 매출 입력]** 버튼 클릭

### 입력 순서

**① 기본 정보 입력**

| 항목 | 내용 |
|------|------|
| 신규/재등 | 신규 회원이면 **신규**, 기존 회원이면 **재등록** 선택 |
| 성함 | 회원 이름 |
| 회원권 종류 | PT, 헬스, 레슨 등 선택 (PT 선택 시 구분이 자동으로 레슨으로 바뀜) |
| 금액 | 결제 금액 입력 |
| 결제 방법 | 현금 / 카드 / 계좌이체 선택 |
| 구분 | **센터** 또는 **레슨** — 어느 매출로 잡을지 선택 |
| 특이사항 | 카톡 보고 문구에만 들어가는 내용 (엑셀에는 기록 안 됨) |

**② [엑셀에 등록] 클릭**

데일리 파일과 총매출 파일에 동시에 자동으로 기록됩니다.  
같은 이름·금액이 오늘 이미 입력되어 있으면 중복 경고가 뜹니다.

**③ 카톡 보고 문구 전송**

등록 완료 후 카톡 보고 문구가 자동으로 만들어집니다.

- **인포방** 버튼: 인포 직원방으로 전송
- **알바방** 버튼: 알바 직원방으로 전송
- **복사** 버튼: 클립보드에 복사 (직접 붙여넣기)

```
📌매출보고📌

신규/재등 : 신규
성함 : 홍길동
회원권 : PT 20회
금액 : 1,200,000원
결제방법: 계좌이체
특이사항:
```

**④ 신규 회원인 경우 — 유입경로 선택**

신규 등록 시 유입경로 입력 창이 추가로 뜹니다.

| 항목 | 내용 |
|------|------|
| 유입경로 | 현수막 / 지인 소개 / 온라인 / 기타 중 선택 |
| 거주지 | 미리 등록된 아파트 단지 목록에서 선택 |

> 유입경로는 월별 보고서에 자동 집계됩니다. 귀찮더라도 꼭 입력해주세요.

## 3. 지출이 생겼을 때

**[🧾 지출 입력]** 버튼 클릭

### 입력 순서

**① 지출 정보 입력**

| 항목 | 내용 |
|------|------|
| 지출 구분 | 지점 비품 / 수수료 / 주차비 / 복리후생 / 운반비 / 당직비 / 홍보 / 회식대 / 간식대 / 구인 / 세금 / 환불 / 기타 |
| 지출 내용 | 구체적인 내용 입력 (예: 청테이프 2개) |
| 금액 | 지출 금액 |
| 결제 방법 | 카드 / 현금 / 계좌이체 |
| 담당자 | 인포 / 트레이너 등 |
| 거래처 | 구매한 곳 |

**② [엑셀에 등록] 클릭**

데일리 파일 지출 시트와 총매출 파일에 동시 기록됩니다.

**③ 카톡 보고 문구 전송**

```
📌지출보고📌

일자: 2026.06.08
구분: 지점 비품
지출내용: 청테이프 2개
금액 : ₩4,960
결제방법: 카드
담당자: 인포
거래처: 삼육오마트
```

## 4. 잘못 입력했을 때

**[📋 내역 조회]** 버튼 클릭

- 오늘 입력한 매출/지출 내역 전체 확인 가능
- 잘못된 항목 선택 → **수정** 또는 **삭제**
- 엑셀 파일을 직접 열지 않아도 됩니다

## 5. 마감 보고할 때

메인 화면의 **오늘 매출 요약** 카드에 현금·카드·계좌·총합이 자동으로 표시됩니다.

**[📋 보고 문구 복사]** 버튼을 누르면 아래 형식의 문구가 클립보드에 복사됩니다.

```
6.8 월

현금:
0원

카드:
336,000원

계좌:
0원

총합:
336,000원
```

> **네이트온 자동 전송이 설정되어 있으면** 평일 23:30, 주말·공휴일 19:30에 자동으로 팀룸에 전송됩니다. 타이틀 바에 `● 자동전송 ON` 이 표시되어 있으면 정상입니다.

## 6. 월 목표 카운트다운 (매일 마감 후)

**[🎯 월 목표 카운트다운]** 버튼 클릭

월 20일 전후부터 매일 마감 보고 후 사용하는 기능입니다.

### 사용 방법

1. 버튼 클릭 → 오늘 데일리 파일의 매출이 자동으로 읽힘
2. 어제까지의 누적 금액에 오늘 매출이 더해진 문구 확인
3. **[저장하고 복사]** 클릭 → 클립보드에 복사됨 → 채팅방에 붙여넣기

> 같은 날 여러 번 열어도 금액이 중복으로 쌓이지 않습니다.  
> **[저장하고 복사]** 를 눌러야 오늘 금액이 저장되고 내일 이어받을 수 있습니다.

## 7. 회원 DB 업데이트 (정기)

회원 현황, 락카 만료 자동 감지, 외국인 회원 상태 갱신 모두 이 작업 후에 반영됩니다.  
**주기적으로** (하루 1회 또는 필요할 때마다) 진행해주세요.

### 방법

1. 브로제이에서 회원 목록 엑셀 내보내기 (`.xls` 파일)
2. **[🔄 회원 DB 업데이트]** 버튼 클릭 → 내보낸 파일 선택
3. 자동으로 처리되고 결과 팝업 표시

   ```
   회원 DB 업데이트 완료
   활성 310 · 만료 85 · 임박 22 · 홀딩 5 · 미등록 3
   ```

4. **새로 락카가 만료된 회원이 생겼으면** 자동으로 팝업이 뜹니다

   ```
   신규 락카 만료자 발생
   홍길동, 김철수 외 1명
   지금 안내 문자를 발송하시겠습니까?
   ```

   → **예** 를 누르면 락카 문자 발송 화면으로 이동

## 8. 락카 만료자에게 문자 보낼 때

**[📱 만료 락카 문자 발송]** 버튼 클릭  
(또는 DB 업데이트 후 팝업에서 **예** 선택)

### 발송 방법

1. 목록에서 문자를 보낼 회원 확인 (기본으로 전체 체크되어 있음)
   - 🌍 표시: 외국인 회원 (한/영 이중 언어로 자동 발송)
   - 체크 해제: 제외할 회원은 체크 풀기
2. 하단 미리보기에서 문구 확인
3. **[📨 선택 회원에게 발송]** 클릭

> 처음 사용 시 **[🧪 테스트 발송]** 으로 내 번호에 먼저 문자가 오는지 확인하세요.

**발송 문구**

- 한국인 회원:
  ```
  [리와인드 휘트니스]
  홍길동님, 안녕하세요!
  락카 이용 기간이 만료되었습니다.
  재계약을 원하시면 센터로 문의해주세요 😊
  ```

- 외국인 회원 🌍 (자동으로 한/영 둘 다 발송):
  ```
  [Rewind Fitness]
  Hi John!
  Your locker rental period has expired.
  Please visit us to renew 😊

  [리와인드 휘트니스]
  John님, 락카 이용 기간이 만료되었습니다.
  재계약을 원하시면 센터로 문의해주세요 😊
  ```

> **문자가 안 보내진다면** → 센터폰 켜져 있는지, SMS Gateway 앱 실행 중인지 확인하세요. ([SMS Gateway 앱 설정 참고](#sms-gateway-앱-초기-설정-최초-1회))

## 9. 매월 1일 — 외국인 회원 공휴일 문자

매월 1일 오전 10시 이후 프로그램이 켜져 있으면 자동으로 팝업이 뜹니다.

### 팝업이 떴을 때

1. 자동으로 감지된 이번 달 공휴일 목록 확인
2. **지방선거일 등 자동으로 안 잡히는 날이 있으면** 직접 추가
   - 날짜: `6/3` 형식
   - 한국어 이름: `지방선거일`
   - 영어 이름: `Local Election Day`
   - **[추가]** 클릭
3. 미리보기에서 발송 문구 확인
4. **[📨 발송하기]** 클릭 → 외국인 활성 회원 전원에게 발송

> 이번 달 공휴일이 없거나 보낼 필요 없으면 **[이번 달 건너뛰기]** 를 누르세요.  
> 다음 달 1일까지 팝업이 다시 뜨지 않습니다.

**발송 문구 예시**
```
[Rewind Fitness / 리와인드 휘트니스]

안녕하세요! 이번 달 공휴일 운영 안내입니다.
Hello! Here is our holiday schedule for this month.

6/6(금) 현충일 — 12시~20시 운영
Jun 6(Fri) Memorial Day — 12PM~8PM
```

## 10. 만료 임박 회원 조회할 때

**[📆 만료 임박 회원 조회]** 버튼 클릭

- 기본값: 70일 이내 만료 예정 회원 목록
- 9일 이하 회원은 **빨간색** 강조
- **[전화번호 복사]** 버튼 클릭 → 전체 번호가 줄바꿈으로 복사됨 → 문자 앱 수신인 칸에 바로 붙여넣기 가능

## 11. 락카 현황 확인할 때

**[🔑 락카 현황]** 버튼 클릭

1. 브로제이에서 회원 목록 내보내기 (`.xls`)
2. 락카 현황 화면에서 파일 불러오기
3. 락카 번호별 그리드 자동 표시

| 색상 | 상태 |
|------|------|
| 초록 | 정상 이용 중 |
| 노랑 | 만료 임박 (10일 미만) |
| 빨강 | 만료됨 |
| 회색 | 비어 있음 |

**만료자 명단 복사**

- **📞 전화번호 복사**: 문자 수신 목록에 바로 붙여넣기용
- **📋 명단 복사**: 자물쇠·물품·연락·락카 회수 체크리스트 형식

## 12. 회원 현황 보고할 때

**[👥 회원 현황 보고]** 버튼 클릭

활성·만료·홀딩·예정·미등록 인원이 카드 형식으로 표시되고, 전일/전주/전월 변화 추세 그래프를 확인할 수 있습니다.

- **[이미지 복사]**: 그래프 포함 이미지를 클립보드에 복사
- **[문구 복사]**: 텍스트 형식 보고 문구 복사

> 회원 현황 데이터는 **[🔄 회원 DB 업데이트]** 를 할 때마다 쌓입니다.

## 13. 문제가 생겼을 때

**매출 요약에 숫자 대신 `—` 가 표시돼요**
→ 메인 화면 `↻` 버튼을 눌러 새로고침하세요. 그래도 안 되면 등록된 데일리 파일이 오늘 날짜 파일인지 확인하세요.

---

**총매출 파일에 이번 달 시트가 없다고 오류가 나요**
→ 매월 첫 결제 시 한 번 뜨는 정상적인 안내입니다. **예** 를 누르면 자동으로 시트를 만들어줍니다.

---

**네이트온에 자동 전송이 안 돼요**
→ 아래를 확인하세요.
1. 타이틀 바에 `● 자동전송 ON` 이 표시되어 있는지
2. ⚙ 설정에 **네이트온 웹훅 URL** 이 입력되어 있는지
3. **프로그램이 켜져 있는지** (종료되면 자동 전송 안 됨)

---

**락카/공휴일 문자가 발송 실패해요**
→ 아래를 확인하세요.
1. **센터폰이 켜져 있고** PC와 같은 Wi-Fi에 연결되어 있는지
2. 센터폰에서 **SMS Gateway 앱이 실행 중** (서버 시작 상태)인지
3. ⚙ 설정의 **센터폰 IP, 포트, 사용자명, 비밀번호**가 앱 설정과 일치하는지

## SMS Gateway 앱 초기 설정 (최초 1회)

센터폰(Android)에서 **Play 스토어** → `SMS Gateway` 검색 → **SMS Gateway for Android** 설치

1. 앱 실행 → **서버 시작** 탭
2. 화면에 표시된 **IP 주소** 와 **포트(기본 8080)** 확인
3. 앱 내에서 **사용자명 / 비밀번호** 설정

그 다음 PC 프로그램 **⚙ 설정** 에 아래 정보 입력:

| 항목 | 내용 |
|------|------|
| 센터폰 IP | 앱에 표시된 IP (예: `192.168.219.206`) |
| SMS Gateway 포트 | `8080` |
| SMS Gateway 사용자명 | 앱에서 설정한 사용자명 |
| SMS Gateway 비밀번호 | 앱에서 설정한 비밀번호 |
| SMS 테스트 번호 | 내 전화번호 (테스트 발송 수신용) |

설정 저장 후 **[🧪 테스트 발송]** 으로 문자 수신 확인하세요.

> PC와 센터폰은 **항상 같은 Wi-Fi** 에 연결되어 있어야 합니다.
