# Gym Ops Automation Program

헬스장 운영 업무 중 반복적인 엑셀 작업과 매출 보고 프로세스를 자동화하기 위한 데스크톱 프로그램입니다.

## 프로젝트 목적

기존에는 매일 수동으로 데일리 엑셀 파일을 생성하고,  
엑셀에서 매출 값을 확인한 뒤 네이트온 보고 문구를 직접 작성하고 있었습니다.

이 프로젝트는 이러한 반복 작업을 줄이기 위해 시작되었습니다.

### 해결하려는 문제
- 다음 날짜의 데일리 파일을 매번 수동 생성해야 함
- 엑셀에서 현금 / 카드 / 계좌 / 총합 값을 직접 찾아야 함
- 매일 같은 형식의 보고 문구를 반복 작성해야 함

## MVP v1 범위

1. 기준 엑셀 파일을 선택한다
2. 버튼 클릭 시 다음 날짜의 파일을 자동 생성한다
3. 사용자가 오늘의 엑셀 파일을 선택한다
4. `데일리매출` 시트의 지정 셀 값을 읽는다
5. 아래 형식의 보고 문구를 자동 생성한다

예시:

```text
3.31 화
현금: 100,000원

카드: 200,000원

계좌: 300,000원

총합: 600,000원
